オセロ Web版
==============

【すぐに動かす方法】
index.html をダブルクリックしてください。
通常のゲーム機能はローカルでも動作します。

※ブラウザのセキュリティ設定によっては、ローカルファイルから音声が鳴らない場合があります。
公開後のURLでは通常どおり鳴ります。

【収録機能】
・シングルプレイ／マルチプレイ
・先手／後手選択
・CPU難易度：かんたん／ふつう／むずかしい／不可能
・Undo
・パス、勝敗判定
・黒白の石数表示
・置けるマス表示
・効果音
・黄色／赤色カーソル
・マウス／キーボード操作
・再戦、メニュー復帰
・スマートフォン向けレスポンシブ表示

【操作】
シングル：マウス、または矢印キー＋Enter
マルチ：黒＝マウス、白＝矢印キー＋Enter
Esc＝取り消し
N＝ゲーム終了後の再戦
Z＝スタート画面へ戻る

【注意】
「不可能」は強い探索CPUですが、数学的な無敗保証はありません。
PCやスマートフォンの性能によっては、終盤の思考に時間がかかる場合があります。
